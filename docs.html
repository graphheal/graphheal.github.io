Getting Started

Install, configure, and embed Graph-Heal in minutes.

Installation

pip install graph-heal

Quickstart

from graph_heal import GraphHeal

gh = GraphHeal()

gh.add_service("frontend")
gh.add_service("backend", dependencies=["frontend"])

gh.update_metrics("frontend", {"cpu_usage": 45, "latency": 120})
Java Quick-Start 
// 1) Pull in our Maven artifact
//    <dependency>
//      <groupId>io.graphheal</groupId>
//      <artifactId>graph-heal-client</artifactId>
//      <version>0.9.0-rc</version>
//    </dependency>

import io.graphheal.GraphHeal;
import io.graphheal.ServiceNode;
import java.util.List;
import java.util.Map;

public class HealthDemo {
    public static void main(String[] args) {
        GraphHeal gh = new GraphHeal();
        gh.addService("frontend");
        gh.addService("db", List.of("frontend"));

        // simulate some metrics
        gh.updateMetrics("frontend", Map.of("cpu_usage", 90.0, "latency", 1200));

        var summary = gh.getHealthSummary();
        System.out.println(summary);
    }
}

Configuration

Graph-Heal ships with sensible defaults but is fully customizable via YAML or code-based configuration. See the examples directory for templates.
