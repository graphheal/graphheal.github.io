Getting Started

Install, configure, and embed Graph-Heal in minutes.

Installation

pip install graph-heal

Quickstart

from graph_heal import GraphHeal

gh = GraphHeal()

gh.add_service("frontend")
gh.add_service("backend", dependencies=["frontend"])

gh.update_metrics("frontend", {"cpu_usage": 45, "latency": 120})

Configuration

Graph-Heal ships with sensible defaults but is fully customizable via YAML or code-based configuration. See the examples directory for templates.
